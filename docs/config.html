Configuration

General

  • Always show tray icon

    • This will make the tray icon always visible, even when the player isn't minimized.

  • Start minimized to tray

    • This will start Simple Amp minimized to the tray.

  • Always ontop

    • This will make all the windows of Simple Amp to always be ontop all other windows.

  • Snap windows to screen edges

    • This will make the player and playlist windows 'stick' to the screen edges.

  • Playlist buttons do default action when clicked

    • This will make the playlist buttons do the default action on their popup menus whenever you leftclick them. Rightclick to show the popup menu instead.

  • Dynamic size playlist columns

    • This will make columns have a larger width if the following column(s) is empty. This can make the playlist look a bit cluttered, and perhaps a bit harder to read. This only applies to the playlist when the playlist window is large.

  • Fade windows

    • This will fade the player and playlist windows in when Simple Amp is started. This will not be seen when you have 'Start minimized to tray' enabled. This is pretty useless, turn off to speed up on slower systems.

  • Tray Icon

    • This slider will let you choose the icon that represents Simple Amp in the system tray.

    Visualizations

  • Visualization update speed

    • This will control how often the visualization part of Simple Amp is updated. One second is 1000 milliseconds. You can divide 1000 with this value to get the frames per second. Setting this value too low will not make the visualization update faster if your computer can't keep up.

  • Disable fade on all presets

    • This will turn off fade on all loaded presets. This will dramatically speed up the visualization.

    File Browser

  • Read Extended File Information

    • Here you can set up when the file browser should read extended information from files you are browsing. Turning this off, speeds up file browsing.

    Media Library

  • Use Media Library

    • This will enable the use of the media library. This will speed up when reading files, but will take more memory.

  • When there are no tags, get artist & title from filename

    • Usually, music with no tags will not be listed under for example the 'Artist' filter in the Media Library Browser. With this option on, though, the browser will attempt to take Artist and Title from the filename of the music.

  • Autosize the list columns to fit their content

    • This will size the columns in the Media Library Browser so that their contents are visible. This might make very wide lists though.

  • Hide modules, midis and wavefiles in the Media Library Browser

    • This will hide all modules, midi-files and wave-files when browsing the Media Library. These file types do not contain much useful information anyways and just clutters up the list. Browsing will also go a bit faster with this on.

  • Library Maintenance

    • This will open the maintenance window which will let you clean up the Media Library. You should do this once in a while if you move/remove files played with Simple Amp often.

    Sound Device

  • Output Engine

    • You should choose Directsound for best performance, but if you experience any problems try Windows Multimedia WaveOut. Use WaveOut under Windows NT.

  • Sound Device

    • Choose your which of your installed soundcards to use. If there are none detected, try another Output Engine.

  • Mixer

    • 'Interpolated MMX P5 16 bit' is suitable for Pentium MMX type computers and 'Interpolated MMX P6 16 bit' for Pentium II or better. 'Interpolated CPU/FPU 32 bit' is the slowest mixer, and is for those without MMX support. In order to use the Surround option 'Interpolated CPU/FPU 32 bit' is required.

  • Sampling Frequency

    • This controls the quality of the sound. 44100 Hz is Audio CD quality and 48000 Hz is professional DAT quality. Choose an lower value to speed up.

  • Max Channels

    • This controls the maximum allowed channels playing at once in modules. Setting this to 128 will not slow down when playing an module that uses 16 channels, it just allows 128 channels at once.

  • Buffer Size

    • More buffer size means more stability, but also means more latency when you seek into a song. So you should set it to the lowest possible that doesn't make the music skip.

  • Enable DSP

    • Checking this box will enable DSP (Digital Sound Processing). DSP will let you view sound visualizations and use sound filters. Disabling DSP will speed up.

  • Enable DirectX 8 Effects

    • Checking this box will enable the ability to use DirectX 8 based effects and the Equalizer. The effects include Chorus, Compressor, Distortion, Echo, Flanger and Gargle. DirectX 8 or higher is required. Unchecking this box will speed up.

    Speakers

  • Speaker Setup

    • Select your speaker configuration here. This will only have effect on soundcards, drivers and speakers that support the selected setup. If you have chosen Windows Multimedia WaveOut as output engine, these settings might not work correctly. Select 'Use Windows Default / No Change' to not setup the speaker configuration, and instead use the one already in effect when starting Simple Amp.

  • Speaker Panning

    • This will pan the music between the left and right speakers. This is useful if one of your speakers are further away from you than the other. Panning will not work well when playing some modules because they can change it as they want.

  • Surround

    • Surround sound is a fake dolby trick that effectively pans the channel to the center, but inverts the waveform in one speaker to make it sound fuller or spacier, or like it is coming out of space between the 2 speakers. Surround works best with an centered panning and stereo speakers. 'Interpolated CPU/FPU 32 bit' mixer setting is required.

  • Pan Seperation

    • Pan seperation affects modules and how much each sound in the left and right speakers are seperated. An high value will make the module sound more 'stereo'.

    File Types

    This section will let you select which file types you want to open with Simple Amp whenever you doubleclick them.

  • Register file types on start

    • This will make Simple Amp register the file types to be associated with Simple Amp every time you start the program. This is useful to prevent other program from 'stealing' the associations.

  • Default action

    • This will set what action will be performed with the associated files.
