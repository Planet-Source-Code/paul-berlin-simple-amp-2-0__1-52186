General Help


Simple Amp works pretty much like any other media player. There are some thigs thay may need some explaining.

  • You can move items in the playlist by dragging them with the mouse.
  • You can drag files to simple amp to add/open the files/folders.
  • You can rightclick anywhere in the player to open the system menu.
  • You can rightclick on the visualization area to load/edit presets.
  • Most sliders in the sound studio and preset editor has tooltips that describe their function.
  • The audio tab in the file viewer will show album covers, if found.
  • You have to enable DirectX 8 Effect in the settings to use some effects in the sound studio.
  • You can doubleclick items in the File Browser and the Media Library Browser to play them without adding them to the playlist.
  • Most functions have keyboard shortcuts. They are displayed in their respective menus.

    Program Command Line Arguments


    When simple amp is running, run 'Simple Amp.exe -next' to play next song in the running instance, etc. Only one command will be accepted (for now).

    This is useful if you have a keyboard with programmable buttons. you can for example set one button to start 'Simple Amp.exe -playpause' to play/pause the music in Simple Amp.

    Note: Only the latest started instance will respond to the commands.

  • -playpause, -pauseplay

    • Plays/Pauses music.

  • -prev, -previous

    • Plays previous item in playlist.

  • -next

    • Plays next item in playlist.

  • -toggleshuffle, -shuffle

    • Toggles Shuffle on/off.

  • -togglerepeat, -repeat

    • Toggles Repeat on/off.

  • -showhide, -hideshow

    • Shows/hides program.

  • -restart, -stopplay

    • Stops music, rewinds, and plays again.

  • -forward, -fwd

    • Move music forward 5 seconds.

  • -back, -bck

    • Move music back 5 seconds.

  • -showopen

    • Shows Browse window.

  • -open [Filename]

    • Opens file [Filename]

  • -ignore

    • Lets you have more than one instance of Simple Amp Running.
