Features


These are the most important features:

Simple Amp is an music player capable of playing the following formats:

  • Mpeg Audio Files (mp3, mp2)
  • Ogg Vorbis Audio Files (ogg)
  • Microsoft Windows Media Audio Files (wma)
  • Microsoft Advanced Sound Format Audio Files (asf)
  • Microsoft Waveform Audio Files (wav)
  • Protracker/Fasttracker Module Files (mod)
  • Screamtracker 3 Module Files (s3m)
  • Fasttracker 2 Module Files (xm)
  • Impulse Tracker Module Files (it)
  • MIDI Files (mid, midi, rmi)
  • DirectMusic Segment Files (sgm)
  • Compact Disc Digital Audio (cda)

    • and can open the following playlist formats:
  • Simple Amp Playlist Files (playlist)
  • M3U playlists
  • PLS playlists

    • The Media Library will keep information of each media file on your harddrive to speed up access to them. You can also browse and search the library in the Media Library Browser.

    The Sound Studio will let you setup equalizer settings and different sound filters, like Chorus, Echo, Flanger and Pitch.

    Several included high quality skins.

    Several Visualizations, fully customizable.

    File Viewer where you can edit tags and view media info, album covers.

    You can control the entire program through command line arguments.

    Check out General Help and Configuration for more info.
