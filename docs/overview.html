Simple Amp 2.0


Simple Amp was written by me, Paul Berlin, initially for personal use because I just wanted a player that I could control from my keyboard. In the beginning it was very simple (thus the name, Simple Amp) but eventually I began improving it.

I began writing Simple Amp sometime in the spring of 2002, and in the beginning of the summer 2002 I released Simple Amp 1.2 to PlanetSourceCode. Compared to the current version it sucked, but it got 20 or so excellent ratings.

I have worked with this version on and off since then (wow, it's been over a year). Continue on to features for some info of exactly how much better and improved Simple Amp 2.0 is. Take a look at the programming notes for some more information.

Please vote for Simple Amp 2.0 at PlanetSourceCode.

Using the fmod sound library, http://www.fmod.org. FMOD is only free to use for non-commercial products!

Feel free to change/add/improve this code, but if you do it would be nice if you send me the updated code =).

You are free to use this code in your own programs, as long as you give credit where credit is due.

 
System Requirements


Minimum system:

  • Microsoft Windows 2000/XP
  • Windows compatible sound card
  • About 15 Mb diskspace (with included skin)
  • Visual Basic 6.0 Runtime files

    • Recommended system:

  • Microsoft Windows 2000/XP
  • High quality sound card
  • About 15 Mb diskspace (with included skin)
  • Visual Basic 6.0 Runtime files
  • 1Ghz or faster computer (for fast visualization when using fade)
  • 1024x768 32-bit or higher screen resulotion
  • Direct X 8.1 (For equalizer & some effects)
